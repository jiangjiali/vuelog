# [Vuelog](https://vuelog.js.org/)
> A backend-free blog system built on top of Vue.js

```
npm install
```

### Compiles and hot-reloads for development

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### Run your unit tests
```
npm run unit
```
